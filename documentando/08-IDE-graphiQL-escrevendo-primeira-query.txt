01) ACESSANDO IDE GRAPHIQL

    ISSO SERA FEITO ATRAVES DA ROTA:

    http://localhost:8000/__graphql

    ACESSANO ESSE ENDERECO IREMOS TER 
    ACESSO A IDE QUE O GATSBY DISPONIBILIZA PARA TRABALHARMOS COM O GRAPHQL

02) REALIZANDO NOSSA PRIMEIRA QUERY COM GATSBY.

    POR DEFAULT O GATSBY JA TRAZ UMA ESTRUTURA CONTENDO 
    DIVERSAS INFORMACOES ACERCA DE TUDO QUE VEIO POR DEFAULT NA INSTALACAO DO GATSBY. 

    A SEGUIR IREMOS REQUISITAR OS DADOS CONTIDOS EM "SITE", 
    POIS IREMOS PEGAR AS INFORMACOES VINDAS DO "siteMetadata", 
    que são as informações que ESTAO DEFINIDAS DENTRO DO ARQUIVO GATSBY-CONFIG.JS, 
    TAIS COMO O TITLE, O AUTHOR E A DESCRIPTION. 

    query MySiteMetadata {
      site{
        siteMetadata {
          author
        }
      }
    }

    AS QUERYS PODEM SER NOMEADAS OU NAO. 
    CASO NAO FOREM NOMEADAS, 
    BASTA COLOCAR-LAS DENTRO DAS CHAVES. 

03) ALEM DO GRAPHIQL NOS TEMOS UMA SEGUNDA IDE A NOSSA DISPOSICAO. 

    PARA PODERMOS UTILIZAR A VERSAO PLAYGROUND DO GRAPHIQL, 
    IREMOS FAZER A SEGUINTE ALTERACAO NO ARQUIVO PACKAGE.JSON:

    DENTRO DOS NOSSOS SCRIPTS, 
    IREMOS CRIAR O SEGUINTE:

    "develop:playground": "GATSBY_GRAPHQL_IDE=playground gatsby develop",

    AGORA BASTA RODAR O SEGUINTE COMANDO NO TERMINAL:

    yarn develop:playground
